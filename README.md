# jogo da velha (python)

jogo da velha feito em python com interface gráfica, tema dark e persistência de dados em banco sqlite.

projeto focado em praticar:
- lógica de programação
- interface gráfica (tkinter)
- banco de dados
- organização de código

## funcionalidades
- interface gráfica com tema dark
- dois jogadores com nomes
- jogo da velha clássico (3x3)
- destaque visual de vitória
- ranking top 5 jogadores
- dados salvos em banco sqlite
- animações simples (hover e feedback visual)

## tecnologias usadas
- python 3
- tkinter
- sqlite3

## como executar
1. tenha o python 3 instalado
2. clone o repositório
3. no terminal, dentro da pasta do projeto, execute:

```bash
py main.py
